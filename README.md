# 🩺 Sistema Experto para Evaluación del IMC

Sistema experto de escritorio desarrollado en **Python** para calcular el Índice de Masa Corporal (IMC), clasificar el estado nutricional de un paciente y generar recomendaciones utilizando un conjunto de reglas.

## 📋 Características

- Cálculo automático del IMC.
- Clasificación del estado nutricional.
- Recomendaciones basadas en reglas.
- Interfaz gráfica desarrollada con Tkinter.
- Almacenamiento de información mediante SQLite.

## 🛠 Tecnologías utilizadas

- Python
- PyQt5
- SQLite
- Qt Style Sheets (QSS)
- Git y GitHub


## 📁 Estructura del proyecto

```
imc_sistema_experto/
│
├── assets/          # Recursos gráficos
├── data/            # Base de datos SQLite
├── logic/           # Lógica del sistema experto
├── ui/              # Interfaz gráfica
├── main.py          # Punto de entrada
├── requirements.txt
└── README.md
```

## 🚀 Instalación

Clona el repositorio:

```bash
git clone https://github.com/DanielR850/SistemaExperto.git
```

Instala las dependencias:

```bash
pip install -r requirements.txt
```

Ejecuta la aplicación:

```bash
python main.py
```
